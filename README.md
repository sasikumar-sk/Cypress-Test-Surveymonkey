The automation suite is designed to run a set of tests on SurveyMonkey's web application, ensuring the correctness and performance of its features, including survey creation, question addition, and survey publishing

**Technologies Used**
Cypress: End-to-end testing framework for the web.
JavaScript (ES6+): For writing the test scripts.
Chai Assertion Library: For writing assertions in tests.
Alura : Test Reporting Tool
Mocha: Test runner used by Cypress to execute tests.
GitHub Actions / Jenkins: For Continuous Integration and Continuous Deployment (CI/CD).
Cypress Dashboard (Optional): For real-time test results and advanced debugging.
